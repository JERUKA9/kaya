Introduction

Kaya is a board game suite for KDE, containing games such as Chess and Shogi, and built upon a powerful plugin system which makes it easily extensible with new games, themes and behaviour.

Features (release 0.1)

  • Fully functional shogi, chess and minichess interface.
  • Game history sidebar.
  • Customizable interface.
  • FICS support: play games, interact using a text console.
  • Load, edit and save chess games.
  • XBoard-compatible chess engine support
